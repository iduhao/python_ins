### 1. 环境

Python版本: `Python 3.x`
部署机器: `173.248.234.200`
代码位置：`/root/CrawlCode`

### 2.运行

1. 境变量（已配置）
```
export PYTHONPATH="/root/CrawlCode:$PYTHONPATH"
```

2. 安装依赖包 (已安装)
```
pip install -r requirements.txt
```

3. 运行代码
```
cd CrawlCode/Crawler/Instagram/
python Instagram.py user_name
```

4. 下载目录
```
CrawlCode/Crawler/Instagram/data/user_name/
```
